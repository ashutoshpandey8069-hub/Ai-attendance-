AI based face recognition attendance system for colleges.

This website lets you capture faces, store student records, auto-generate barcode ID cards, mark attendance, and review student attendance performance in one working demo.

Project flow
Capture → Match → Mark → Analyze
Face capture: Uses webcam to take a student image and stores a reference profile.
ID card: Generates an English ID card with student details and barcode automatically.
Performance: Shows attendance percentage, trends, and student ranking.
